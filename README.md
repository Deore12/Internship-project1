
Project Overview
Participants act as a junior data analyst working for a fictional client, Daikibo, to improve their manufacturing operations using telemetry data. The simulation takes approximately 1–2 hours to complete and focuses on these key tasks: 

Task 1: Data Unification & Forensic Analysis
Goal: Clean and categorize "messy" forensic data using Excel.
Action: Identify issues in client data, perform basic classification, and support investigative decision-making.

Task 2: Data Visualization (Tableau)
Goal: Create a dashboard to answer specific operational questions.
Action: Analyse telemetry data to find which factory locations have the most machine breakdowns and which specific machines are most prone to failure.

Task 3: Development Proposal
Goal: Translate data findings into a business solution.
Action: Draft a concise software development proposal for a real-time manufacturing status dashboard with smart alerts. 

Key Skills & Tools
Tools: Excel (Forensic analysis) and Tableau (Data visualization).
Skills: Data cleaning, business thinking, dashboard creation, and professional proposal writing.
